# Team 4 - Gym Membership Tracking System

## Project Description
A fitness center system designed to track up to 25 gym member records, managing operational data and membership statuses.

## Team Members & Roles
* **Juan Cervantes** - Project Manager
* **Alicia Espana** - Documentation Manager
* **David Rodriguez** - Software Developer
* **Kevin De La Rosa** - GitHub Coordinator

## System Features & Goals
* Track up to 25 Member Records (ID, Name, Status).
* Manage Status Values: Active, Expired, Suspended.
* Generate Summary Reports (Total, Active, Expired, Suspended members).

## Expected Inputs
* Each member record includes:
  ﻿*﻿ Member ID (unique identifier).
  ﻿﻿* Member Name.
  ﻿﻿* Phone Number.
  ﻿﻿* Email Address.
  ﻿﻿* Home Address.
  ﻿﻿* Date of Birth.
  ﻿﻿* Membership Status (Active / Expired / Suspended).

## Expected Outputs
﻿﻿* Individual member record details.
﻿﻿* Search results matching a Member ID.
﻿﻿* Full membership record list.
﻿﻿* Summary report with totals by status.
